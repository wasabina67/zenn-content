# Dockerイメージ

## イメージを取得

```bash
docker pull hello-world:latest
```

```bash
docker pull python:3.10-slim
```

## イメージ一覧を表示

```bash
docker images
```

## イメージを削除

```bash
docker rmi hello-world:latest
```

```bash
docker rmi python:3.10-slim
```
