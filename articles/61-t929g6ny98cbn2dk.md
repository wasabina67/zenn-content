## Base64 エンコードとは

バイナリデータを ASCII 文字列に変換する方式。メールの添付ファイルや画像の埋め込みなどで使われる。

**使用文字**: `A-Z`, `a-z`, `0-9`, `+`, `/` (64種類) + パディング用の `=`

## Base64 URL エンコードとは

Base64 を URL で安全に使えるように改良したもの。

**変更点**:

| Base64 | Base64 URL |
| ------ | ---------- |
| `+`    | `-`        |
| `/`    | `_`        |
| `=`    | 省略可     |

`+` と `/` は URL で特別な意味を持つため、別の文字に置き換えている。

## Python での例

```python
import base64

data = b"Hello, World!"

# Base64
encoded = base64.b64encode(data)
print(encoded)  # b'SGVsbG8sIFdvcmxkIQ=='

# Base64 URL
encoded_url = base64.urlsafe_b64encode(data)
print(encoded_url)  # b'SGVsbG8sIFdvcmxkIQ=='
```

## 使い分け

- **Base64**: メール、データ URI スキーム
- **Base64 URL**: JWT、URL パラメータ、ファイル名
