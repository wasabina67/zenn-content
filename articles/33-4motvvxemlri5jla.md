## 1. 現在のディレクトリに解凍

```bash
unzip file.zip
```

## 2. 特定のディレクトリに解凍

```bash
unzip file.zip -d /path/to/directory
```

## 3. 進行状況を表示

```bash
unzip -v file.zip
```

## 4. 解凍しないで中身を確認

```bash
unzip -l file.zip
```
