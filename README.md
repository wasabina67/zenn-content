# Zenn CLI
Zenn連携用リポジトリ

#### 👇  新しい記事を作成する

```bash
npx zenn new:article
```

#### 👇  新しい本を作成する

```bash
npx zenn new:book
```

#### 👇  投稿をプレビューする

```bash
npx zenn preview
```

```bash
bash count_articles.sh
```

#### 👇  pre-commit

```bash
pre-commit install
```
